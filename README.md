# Online-Sicher.de — Fake Shop Blocklist

Curated blocklist of confirmed fake shops targeting German-speaking consumers. Updated regularly. Machine-readable formats for DNS blocklists and security tools.

## Files

| File | Format | Description |
|------|--------|-------------|
| `domains.txt` | Plain text, one domain per line | For DNS blocklists (Hagezi, Pi-hole, AdGuard, etc.) |
| `domains.json` | JSON | For API integrations and security tools |

## Usage

**Plain text (DNS blocklists):**
```
https://raw.githubusercontent.com/kdkiss/online-sicher-blocklist/main/domains.txt
```

**JSON:**
```
https://raw.githubusercontent.com/kdkiss/online-sicher-blocklist/main/domains.json
```

## Source

Domains are identified and verified by [Online-Sicher.de](https://online-sicher.de) — a German consumer protection site helping victims of online fraud, phishing, and fake shops.
